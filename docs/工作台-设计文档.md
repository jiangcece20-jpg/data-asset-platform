# 资源管理 — 工作台设计文档

**关联文档：** 资源管理模块产品验收见 `docs/product/2026-04-04-资源管理-模块PRD.md`；权限申请与审批链路见 `docs/product/2026-04-04-权限中心-模块PRD.md`（「我的」见 `2026-04-04-我的-模块PRD.md`）。

## 1. 定位

工作台是资源管理模块的子面板，面向数据开发/数仓工程师，作为每日工作的导航中枢。

核心原则：**只做聚合、提醒、跳转，不做具体操作流程。**

## 2. 入口

资源管理左侧边栏第一项，排在"资源列表"上面。

## 3. 功能边界

| 模块 | 职责 | 与工作台的关系 |
|---|---|---|
| 工作台 | 聚合待办 + 异常提醒 + 资源动态 | 导航中枢，点击跳转到对应模块 |
| 资源列表 | 资源增删改查、上下架、编辑 | 工作台"待上架"点击后跳转到这里 |
| 目录管理 | 目录树维护 | 工作台"目录悬空"点击后跳转到这里 |
| 我的 | 历史行为和收藏的完整记录 | 工作台"我的申请"只显示进行中，"查看全部"跳转到我的 |
| 权限中心 | 权限申请和审批的完整流程 | 工作台「待审批」点击后跳转到**权限中心**（或内嵌审批路由，与 `page-permission` 合并策略见权限中心模块 PRD） |

## 5. 异常资源类型

| 类型 | 说明 | 操作 |
|---|---|---|
| 目录悬空 | 挂载的目录节点已删除 | 跳转重新分配目录 |
| 上游断裂 | 血缘上游表已下线 | 跳转查看血缘 |
| 长期未更新 | 超过 N 天无数据写入 | 确认是否下线 |
| 字段漂移 | 物理字段与元数据不一致 | 跳转同步字段 |
| 权限孤岛 | 负责人已离职无人维护 | 跳转认领/交接 |

## 6. 交互规则

- 所有待办条目点击后跳转到对应模块，不在工作台内操作
- 异常资源红色高亮，置顶显示
- "我的申请"只显示进行中的前 3 条，"查看全部"跳转到"我的"页面
- 统计卡片数字可点击，跳转到对应筛选视图
- 审批流标注"对接飞书审批"（后端对接预留）
