## 2. 模块定位与背景

### 2.1 一句话定位

**我的**是普通用户的个人中心：提交权限申请单、查看我的权限申请进度、管理收藏、查看已授权资产、管理我负责的资源。

### 2.2 非目标

- 替代资源管理后台的批量治理、审核队列（仅提供入口跳转）。
- 承载全量工单查询与审批处理（归审批中心）。
- 承载审批模板配置与全局审批记录（归审批管理）。

## 4. 功能需求

### 4.1 导航切换（FR-MY-01）

- 点击侧栏切换面板，互斥高亮；切换时触发对应面板渲染。
- 默认从一级导航进入「我的」页面时，优先落在「我收藏的」面板。
- 带有明确业务上下文的深链进入时，应优先落在指定面板；指定面板优先级高于默认落点。
- 若用户是在详情页、资源发现、审批中心重新申请等申请权限链路中点击「去申请单提交」或等效动作，则进入「我的」页面后直接打开「申请单」面板。
- 若用户是从工作台或个人进度入口查看申请记录，则进入「我的」页面后优先打开「我申请的」面板。
- **验收**：无重复渲染导致闪烁；保留各面板筛选状态（产品可约定是否重置）；默认入口与申请链路入口落点符合预期。

### 4.2 申请单（FR-MY-02）

【此处需插入原型图：「我的」— 申请单面板，展示已选资产列表（含移除按钮）+ 统一申请理由输入框 + 提交按钮的购物车式布局】

- **入口**：从资源发现卡片、资产目录、统一详情页等处点击「申请权限」或「加入申请单」，将资产加入申请单；当用户从申请权限链路点击「去申请单提交」或从审批中心点击「重新申请」时，应直接进入本面板。
- **动作区分**：
  - **加入申请单**：仅将资产加入待提交列表，不强制离开当前页面。
  - **去申请单提交**：跳转到「我的 > 申请单」，聚焦当前待提交资产并继续填写理由。
- **展示**：已选资产列表，每行展示资产名称、类型、来源，支持单条移除和一键清空。
- **统一申请理由**：必填，字数校验（≥10 字，策略可配置）。
- **提交**：生成申请单/子单，提交后 toast 提示「申请已提交」，申请单自动清空。
- **去重**：同一资产重复加入时 toast 提示「该资产已在申请单中」，不重复添加。
- **角标**：左侧导航「申请单」入口展示当前待提交资产数量角标。
- **验收**：重复加入去重；提交后申请单清空；理由校验生效；空申请单展示引导文案「暂无待申请资产，去资源发现浏览」。

### 4.3 我收藏的（FR-MY-03）

【此处需插入原型图：「我的」— 我收藏的面板，展示收藏列表（名称/类型/来源/目录/负责人/收藏时间/取消收藏按钮）及筛选条】

- **数据**：资产与资源收藏合并列表；支持类型、来源平台筛选与关键词搜索。
- **操作**：行内进入详情页；**取消收藏**持久化并刷新列表。
- **验收**：空状态文案；取消收藏后即时从列表移除。

### 4.4 我申请的（FR-MY-04）

- **定位**：仅展示当前用户发起的**权限申请**进度，不含治理操作申请（上架/下架/目录变更等治理类工单在审批中心「工单查询」中查看）；作为普通用户查看个人权限申请进度的快捷视图。
- **数据**：当前用户发起的权限申请列表。
- **列表字段**：资产名称（可点进详情页）、资产类型 tag、申请时间、状态 tag、工单（展示工单编号并可跳转审批中心工单详情，所有状态均展示）。
- **状态 Tab**：全部 / 审批中 / 已通过 / 已拒绝 / 已撤回。
- **筛选**：资产名称关键词搜索。
- **操作**：审批中状态可撤回；已拒绝可重新申请。
- **跳转**：底部提供「查看全部工单」链接，跳转审批中心「工单查询」页面。
- **验收**：列表仅展示权限申请类型，不出现治理操作申请；撤回后即时更新；空状态文案清晰。

### 4.5 我有权限的（FR-MY-05）

- **数据**：当前用户已授权对象列表；权限类型、生效时间、授权工单（展示工单编号并可跳转审批中心工单详情）。
- **操作**：查看详情。
- **验收**：与即席查询「有权限表」、详情页「我的权限」口径一致。

### 4.6 我负责的（FR-MY-06）

【此处需插入原型图：「我的」— 我负责的面板，展示资源列表（含类型/状态/角色筛选）+ 批量选择 + 批量交接负责人弹窗 + 「前往资源管理」引导入口】

- **数据**：技术负责人或业务负责人匹配当前用户的资源。
- **筛选**：类型、状态、角色（技术/业务/双负责人）。
- **批量**：多选、全选、批量交接负责人（弹窗操作）；单行交接。
- **引导**：「前往资源管理」前往资源管理页。
- **验收**：批量条数与按钮显隐正确；非负责人不可见对应行。
