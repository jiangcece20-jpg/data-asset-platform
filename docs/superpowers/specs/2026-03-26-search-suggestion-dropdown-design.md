# 搜索下拉建议层设计说明

**目标**

优化资产检索页顶部搜索框，在用户输入关键词时实时展示下拉命中结果。下拉层需要同时支持中文名和英文名命中，并在候选项与底部操作栏中给出明确的操作提示。

**交互设计**

- 输入框获得焦点且存在输入内容时，显示搜索建议层。
- 建议层优先展示命中的资产和资源候选，支持中文名、英文名、别名和简介命中。
- 候选项展示以下信息：
  - 英文技术名
  - 中文展示名或功能说明
  - 类型、负责人
  - 命中来源提示，如“命中英文名”“命中中文名”
  - 当前操作提示，如“Enter 查看详情”
- 键盘操作支持：
  - `↑ / ↓` 切换高亮候选
  - `Enter` 打开当前高亮候选；若无候选，则执行全文搜索
  - `Esc` 收起建议层
- 鼠标悬停候选时同步切换高亮，点击候选直接打开对应详情页。
- 点击输入框外部区域时关闭建议层。

**范围约束**

- 本次不实现 AI 搜索切换。
- 保留原有“搜索”按钮和全文结果页逻辑。
- 仅增强输入中的快捷命中体验与操作提示。
