# 来源徽标展示设计说明

**目标**

在资源发现列表和详情页表头中，以“来源徽标 + 资源名”的方式展示来源信息，让用户在第一眼就能识别资源来自哪个系统及对象类型。

**展示方案**

- 在资源发现列表名称前增加来源徽标
- 在详情页表头名称前增加同样的来源徽标
- 来源徽标文案格式为：`来源系统 + 资源类型`
  - 例如：`MaxCompute表`
  - `SelectDB表`
  - `业务库表`
  - `万联灵析报表`

**视觉规则**

- 来源徽标：
  - 小字号
  - 浅底色
  - 根据来源类型区分颜色
- 资源英文名：
  - 主标题样式
- 资源中文名：
  - 作为副标题，字号更小、颜色更浅

**实现范围**

- 本次只改资源发现列表和详情页表头
- 不改其它页面的卡片或搜索建议层
